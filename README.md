# DonjonFlash

**Action RPG tactique top-down avec donjons proceduraux et audio chiptune.**

Joue sur : [games.hylst.fr/donjonflash/](https://games.hylst.fr/donjonflash/)

## Comment jouer

1. Rends-toi sur [games.hylst.fr/donjonflash/](https://games.hylst.fr/donjonflash/)
2. Choisis ta classe de hero (Guerrier, Ranger ou Filou)
3. Explore les salles procedurales
4. Vaincs les ennemis et recolte le butin
5. Trouve la cle d'or pour ouvrir la porte de sortie
6. Avance au niveau suivant !

## Classes de heros

| Classe | PV | Vitesse | Arme | Sort |
|--------|-----|---------|------|------|
| Guerrier | 7 | Moderee | Epee lourde rotative | Boules de Feu |
| Ranger | 5 | Rapide | Arc long (fleches perforantes) | Nova de Gel |
| Filou | 4 | Maximale | Eventail de dagues | Boules de Feu |

## Controles

- **Fleches / ZQSD** : Deplacement
- **Espace** : Attaque principale
- **Shift / F** : Sort secondaire (parchemin)
- **Echap / P** : Pause
- **Mobile** : D-pad tactile + boutons d'action
