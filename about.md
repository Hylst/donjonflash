# A propos de DonjonFlash

**DonjonFlash** est un Action RPG tactique top-down开发é par **Geoffroy Streit** (alias **Hylst**).

## Le jeu
DonjonFlash est un dungeon crawler frenetique avec :
- 3 classes de heros avec des playstyles uniques
- Salles proceduralement generees (6 types d'architecture)
- Systeme de loot (potions, parchemins, nourriture)
- Ennemis varies (rapides, tank, tireurs)
- Audio chiptune procedural (Web Audio API)
- Rendu Canvas 2D avec eclairage dynamique

## Le createur
- Developpeur d'applications, ancien graphiste et compositeur
- Adepte de retrocomputing (Atari ST) et de demoscene
- 49 ans, un geek de la premiere heure

## Stack technique
- React 19 + TypeScript 5.9 + Tailwind CSS 4
- Vite 7 + vite-plugin-singlefile (build single-file)
- Canvas 2D pour le rendu
- Web Audio API pour l'audio procedural
- 100% frontend, zero backend
